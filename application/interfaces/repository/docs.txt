#
Описание интерфейсов репозитория

#
Интерфейсы пишем через префикс i и с большой буквы имя сущности
Без индексов. 

#
В интерфейсах репозитория описываем базовый CRUD для работы сервисов с сущностью

# 
Репозиторий это абстракция над какой либо базой данных или аналога или микросервиса для работы с сущностью

#
Querry - опциональный параметр, который определяет по какому полю ищут сущность, а так же пагинацию. 
По необходимости можно добавить дополнительные параметры для фильтрации или поиска.  
    Например, если сущность имеет поле isDeleted : boolean
