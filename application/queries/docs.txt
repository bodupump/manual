#
Querry параметр 
Добавляется в запрос оп поиску сущности.
Определяет по какому полю искать сущность - Target
И значение, по которому ищут сущности - Value
При множественном поиске присутствует пагинация - offset - страница поиска и limit - количество на странице
Все параметры необязательные
#
Наследуется от BaseOneQuerry с дженериком T - Target
