#
ИНФРОСТРУКТУРА

# 
Что является инфроструктурой? Любые сторонние сервисы
- Базы данных и ORM (Postgress + TypeOrm, Radis, ClickHouse, MongoBD + mongoose)
- Брокеры сообщение (Kafka, RabbitMQ)
- Файловые хранилоща (S3)
- Стороннее апи
- Абстракции над микросервисами

# 
Обязательно в application должен быть интерфейс для инфраструктурного объекта
В appliction ничего из инфроструктуры не экспртируется
Из инфраструктуры экспортируется только класс реализации интерфейса application в вышестоящий слой, например presentation
